# College Launchpad

A web app designed to help high school students plan for college by providing tools for GPA calculation, college discovery, and scholarship filtering.

Live site: https://section33-ops.github.io/college-launchpad/

## Overview

College Launchpad is my first full HTML/CSS/JavaScript project. It was built while learning JavaScript fundamentals and focuses on applying concepts in a real-world style application.

The goal is to give students a simple starting point for understanding their GPA, exploring colleges, and finding scholarships based on academic performance.

## Features

### GPA Calculator

- Add courses with grades and credit hours
- Automatically calculate cumulative GPA
- Input validation to prevent empty entries
- Dynamic table updates using JavaScript

### College Finder

- Displays a list of colleges from a JavaScript data file
- Basic filtering and rendering using DOM manipulation

### Scholarships Page

- Displays scholarships with GPA requirements
- Structured data rendered dynamically into a table

### UI / UX

- Multi-page layout
- Responsive design for mobile and desktop
- Clean navigation system
- Modular JavaScript structure
